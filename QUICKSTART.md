# 快速启动指南

## 🚀 5分钟快速上手

### 步骤1: 安装依赖

```bash
pip install -r requirements.txt
```

如果遇到网络问题：
```bash
pip install -r requirements.txt -i https://pypi.org/simple
```

### 步骤2: 启动Web界面

```bash
streamlit run web_app.py
```

浏览器会自动打开，或手动访问：http://localhost:8501

### 步骤3: 运行第一个回测

1. 在Web界面左侧选择：
   - 数据源：akshare（默认，无需配置）
   - 股票代码：000001（平安银行）
   - 开始日期：2023-01-01
   - 结束日期：2024-01-01

2. 选择策略：双均线（默认）

3. 点击"开始回测"

4. 查看结果和图表

## 📋 常用功能

### 命令行回测
```bash
python main.py
```

### 参数优化示例
1. 打开Web界面
2. 选择"参数优化"页面
3. 设置参数范围（如：短期均线 5,10,15,20）
4. 点击"开始优化"
5. 查看最优参数和Top 10结果

### 实时信号生成
1. 打开Web界面
2. 选择"实时信号"页面
3. 输入股票代码
4. 选择策略
5. 点击"生成信号"

## 🔑 数据源配置

### akshare（默认，推荐）
- 无需配置，直接使用
- 支持A股数据
- 免费使用

### tushare（可选）
- 需要注册账号：https://tushare.pro/
- 获取token后，在Web界面侧边栏输入
- 数据更全面，但需要积分

### yfinance（可选）
- 无需配置
- 支持美股、港股等
- 股票代码格式：AAPL（美股）、0700.HK（港股）

## 💡 使用技巧

1. **首次使用建议**：先用akshare数据源和双均线策略熟悉系统
2. **参数优化**：不要设置太多参数组合，避免计算时间过长
3. **风险控制**：建议启用止损，设置5-10%的止损比例
4. **策略组合**：可以组合多个策略，提高稳定性

## ❓ 常见问题

**Q: 数据获取失败？**
A: 检查网络连接，或尝试切换数据源

**Q: Web界面打不开？**
A: 确保已安装streamlit：`pip install streamlit`

**Q: 参数优化很慢？**
A: 减少参数组合数量，或使用更短的时间范围

**Q: 如何添加自己的策略？**
A: 参考README中的"扩展开发"部分，继承BaseStrategy类
